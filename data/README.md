# data/ folder

Put the quarterly source files here, named exactly:

- `LCA_Disclosure_Data_FY2026_Q2.xlsx`  (DOL FLAG quarterly LCA file — keep original name)
- `Approvals_Receipts_MA.xlsx`          (USCIS Data Hub — Massachusetts)
- `Approvals_Receipts_National.xlsx`    (USCIS Data Hub — all states)

When you upload or change a file here, the GitHub Action rebuilds the
dashboard automatically. See `AUTOMATION.md` in the repo root for full
instructions.

Replace these files each quarter — only the latest quarter is needed.
